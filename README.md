#GETIP

Command line utility to get location information for an IP or Domain name via [ip-api.com](http://ip-api.com/docs/api:json)

## Usage

Add binary to $PATH

```
$ getip google.com

Looking up: www.google.com

| Querying IP Address (You)      | 173.194.112.114     |

```
